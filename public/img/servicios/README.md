# Imágenes de landings de servicios

Cada landing en `/servicios/[slug]` espera una imagen vertical aquí.

## Archivos esperados

| Slug / Landing | Archivo |
|---|---|
| `/servicios/quirofano` | `quirofano.jpg` |
| `/servicios/ginecologia` | `ginecologia.jpg` |
| `/servicios/neonatologia` | `neonatologia.jpg` |
| `/servicios/cuidados-intensivos` | `cuidados-intensivos.jpg` |
| `/servicios/laboratorio` | `laboratorio.jpg` |
| `/servicios/centro-imagenes` | `centro-imagenes.jpg` |
| `/servicios/emergencia` | `emergencia.jpg` |
| `/servicios/hospitalizacion` | `hospitalizacion.jpg` |
| `/servicios/farmacia` | `farmacia.jpg` |
| `/servicios/biologia-molecular` | `biologia-molecular.jpg` |
| `/servicios/cafeteria` | `cafeteria.jpg` |

**Tamaño recomendado:** 900×1200 px (vertical / aspecto 3:4). JPG o PNG.

Sube los archivos con el nombre exacto y la landing los muestra automáticamente.
Si una imagen falta, la landing muestra el placeholder con el gradiente de marca y
el nombre del servicio (resistente a 404 desde v2 de `PlaceholderImage`).
